# offsec-writeups

A personal collection of cybersecurity writeups, study notes, and learning documentation built while working through platforms like TryHackMe, HackTheBox, and various CTFs.

These aren't polished blog posts — they're structured study notes written to actually stick. Each writeup covers the key concepts, terminology, workflows, and real-world relevance of the topic, written in a way that's useful to come back to.

## What's Inside

Each writeup follows a consistent structure:

- **Overview** — what the room covers and why it matters
- **Key Concepts** — the important ideas explained properly
- **Terminology** — definitions worth having in one place
- **Practical Examples** — commands, payloads, and walkthroughs where applicable
- **Workflow / Process** — communication flows and step-by-step processes
- **Real-World Relevance** — where this shows up in actual security work
- **Key Learnings** — the takeaways worth remembering
